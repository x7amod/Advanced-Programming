## First Time Setup
### !! Note If you already have the old DB file created, remove it first. !!
1. Open `TrainingInstitute.sln`
2. Right Click On "Solution 'TrainingInstitue' (3 of 3 Projects)", choose Restore NuGet Packages (to install packages).
3. Open NuGet Package Manager Console (Toolbar above -> Tools -> NuGet Package Manager -> NuGet Package Manager Console)
4. Run `Update-Database` to create the database
5. Test by running the project (verify no build errors)
